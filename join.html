OZO + CJONE 통합회원 가입

CJ ONE 상세 혜택

  • 기본 적립율
    • 유료 영화관람 금액의 3 ~ 7% 적립
    • 매점 결제 금액의 0.5% 적립
      • 적립 방법 : 영수증 하단의 적립번호와
        인증번호를 OZO 홈페이지/모바일에서 입력
      • 이벤트 기간 중에만 적립 가능
        (일부매장 제외)
  • 제휴 할인 시 / 적립 가능매장
    • 제휴할인 + 결제금액의 0 ~ 7% 적립
  • 기타 적립안내 / 포인트 사용단위
    • 단체할인 시, 무비머니, 관람권,
      상품권 등으로 구매/결제 시 적립 제외
    • 최대 1일 6회(현장 또는 온라인) 또는 1일
      24매(온라인) 예매제한
    • 사용 단위
      • 매표, 매점 : 500P
      • 온라인 예매 : 10P

※ CJ ONE 통합회원으로 가입되더라도 정보 제공 및 약관 동의가 되지 않은 CJ ONE 제휴 브랜드에는 개인정보가 제공되지 않습니다.
